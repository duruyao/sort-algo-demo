# Sort Algorithm


## Directory

```shell
.
├── Makefile
├── obj
│   ├── sort_algo.o
│   └── test.o
├── README.md
├── run
└── src
    ├── sort_algo.c
    ├── sort_algo.h
    └── test.c
```

## Content
The source code contains of the following sort algorithm.

- insert sort
    - base pointer
    - base value

- select sort
    - base pointer
    - base value

- bubble sort
    - base pointer
    - base value

- heap sort base pointer

- quick sort base pointer
    - using k-medium
    - using BFPRT

- bucket sort base pointer

- 2-way merge sort base pointer

- shell sort base pointer

- BFPRT algorithm

## Usage

Compile source code.

```shell
$ make
```

Run executable file.

```shell
$ ./run
```

Clear object files and executable file.

```shell
$ make clear
```
